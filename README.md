# Salesy – Business Management SaaS

Salesy is a full-stack web application designed to help small businesses manage **sales, inventory, customers, and invoices** in a simple and efficient way.

## Key Features

* Dashboard with sales analytics
* Customer & inventory management
* Invoice generation with downloadable PDF
* QR-based product labeling
* Stock tracking with real-time updates
* Mobile-responsive UI

## Tech Stack

* **Frontend:** Next.js, React, Tailwind CSS
* **Backend:** Next.js API Routes, MongoDB, Mongoose
* **Auth:** NextAuth.js
* **Charts:** Recharts
